# Machine-Learning-AI-Learning-Insight
Ini merupakan notebook untuk model AI Learning Insight yang telah kami buat. Model ini mengklasifikasi user ke dalam beberapa kelompok learner, yaitu *fast*, *reflective*, *consistent*, dan *struggling* berdasarkan metrik: `avg_completion_time`, `total_modules_read`, `avg_exam_score`, `login_frequency`, dan `failed_exams`.
